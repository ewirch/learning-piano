# Common NOTE ≠ glued FINGER: a shared tone doesn't guarantee a stationary finger

Lesson 2 originally taught that every common tone gives a "glued finger" handrail.
Eduard corrected this: when a chord's interval shape redistributes the fingers, the
shared note stays on its key but the **finger** changes — so it's a re-fingered
landmark, not a don't-lift handrail. This sharpens (does not replace) the Lesson-1
"stationary finger" idea, and must be respected wherever fingering is taught.

**Update 2026-06-10:** the loop was later re-voiced to first inversions (see
[[0003-revoiced-loop-for-voice-leading]]) to remove a V→vi parallel fifth. In that
voicing every shared note keeps its finger, so the loop no longer demonstrates a
held-note-re-finger. The principle below still holds for voicings in general and
stays in the GLOSSARY; the bullets describe the **original** voicing.

## What was true in the original voicing (both hands)
- **C→G**: G shared, same finger both hands → true glued finger.
- **G→Am**: nothing shared → block-shift up one white key, fingers unchanged.
- **Am→F**: A glued both hands; C glued in LH (finger 3) but re-fingered in RH (3→2).
- **F→C**: note C shared but NO glued finger — RH 2→3, LH 3→2.

(Fingering: RH C/G/Am = 1·3·5, F = 1·2·5; LH G/Am/F = 5·3·1, C = 5·2·1.)

## The generative rule behind the fingering (Eduard's)
Keep fingers **3 and 5 within a third** of each other; the **thumb (1)** stretches
to cover any **fourth**. When the fourth lands on the *pinky* side — out of the
thumb's reach — the middle note moves to **finger 2** so 2–5 spans it. The fourth
sits on the pinky side in RH 1st inversion (→ F = 1·2·5) and LH 2nd inversion
(→ C = 5·2·1); everywhere else it's plain 1·3·5 / 5·3·1. Verified: this rule
reproduces all eight loop fingerings exactly. Use it to finger any new close-triad
voicing in future lessons, rather than memorising tables.
