## Canonical voicing for the loop (use this in lesson 3, don't re-pick)
Re-voiced 2026-06-10 to first inversions so V→vi (G→Am) no longer makes parallel
fifths — classic voice leading; see learning-record 0003. Lesson text does NOT
explain this (Eduard's call: irrelevant to the lesson's goal) — only chord FORMS
changed.
Voiced for ≤1-white-key motion per change, all in the B3–A4 box:
  C = C E G · G = B D G · Am = C E A · F = C F A
Fingering (low→high) — DIFFERS PER HAND:
  RH: C 1·3·5 · G 1·2·5 · Am 1·2·5 · F 1·3·5
  LH: C 5·3·1 · G 5·3·1 · Am 5·3·1 · F 5·2·1
  RULE (Eduard's): keep fingers 3 and 5 within a 3rd; thumb takes any 4th. When
  the 4th is on the pinky side, middle note → finger 2 (2–5 spans it). That gives
  RH G & Am = 1·2·5 (4th on top) and LH F = 5·2·1 (4th on bottom); everything else
  is plain 1·3·5 / 5·3·1. Use this to finger ANY new voicing.
Glued FINGER at each change (this voicing glues every shared note):
  C→G : G shared, same finger both hands (RH 5 · LH 1) → glued. Ride it.
  G→Am: shares nothing → block-shift up one white key, fingers unchanged.
  Am→F: C (RH 1 · LH 5) AND A (RH 5 · LH 1) both shared, same fingers → two glued.
  F→C : C shared, same finger (RH 1 · LH 5) → glued.
The note≠finger distinction (learning-record 0002) still holds in general, but this
voicing happens not to trigger it — kept in the GLOSSARY as a principle only.
Lesson 3's LH bass goes UNDER these exact shapes/fingerings — keep them so we
build straight on top.
Key insight (still true): all four chords are white-key only, so the black/white
tactile fingerprint can't distinguish them — orientation comes from the glued
finger (every change but G→Am) and sensing the black-key groups around the hand.

## Canonical LH bass for lesson 3+ (don't re-pick)
Lesson 3 DROPS the LH triad and plays a single bass ROOT under the RH chords,
~one octave below the B3–A4 box: roots C · G · A · F (the chord roots).
Bass motion round the loop: C→G up a 5th · G→A up a step · A→F down a 3rd ·
F→C down a 4th. Found by Lesson-1 landmarks: C & F = left edge of a group
(2-group, 3-group); G & A = tucked between the 3-group blacks (G first pair, A
last pair). Finger: thumb (1) for accuracy OR pinky (5) for depth — learner picks one.
Coordination principle taught: at C→G, Am→F, F→C the RH is (near) still → use it
as the anchor while the LH leaps; only G→Am has no still hand → both hands rise
one white key together. Deliberately NOT taught yet (scope): root+5th / octave /
broken-bass patterns — flagged only as the next step. Lesson 4 keeps THIS bass +
THESE chords; only the gaze changes (to the page).
