Reference card · Naming seventh chords

Reading the Stack

A seventh chord = a triad + one more third on top. In root position it's a snowman: four noteheads stacked in thirds, all on lines or all on spaces. Read the bottom note, skip up the alphabet — the chord spells itself.

The two pictures

Treble clef = a stylised G; its curl wraps the 2nd line from the bottom, so that line is G. Lines (bottom→top): E G B D F. Spaces: F A C E.

Both directions — same skip, run each way

Read  — staff → name

  1. See a four-high stack → it's a seventh chord.
  2. Name the bottom note off the clef → that's the root.
  3. Alphabet-skip up ×3 for the 3rd, 5th, 7th.

Write — name → staff

  1. Put the root notehead on its line or space.
  2. Root on a line → all lines; on a space → all spaces.
  3. Fill the next three same-kind slots going up.

All naturals (no sharps/flats yet). On the keyboard each is every other white key — play it to check your reading. *D's root sits in the space just below the staff; C's root is middle C, on one ledger line below.

The whole card in one line: four notes stacked tight → seventh chord; name the bottom note → root; skip up the alphabet for the rest — all on lines or all on spaces, never mixed.

This card is the skeleton — letter-names only. What turns C·E·G·B into a major-7, a dominant-7, a minor-7, or a diminished chord is a matter of sharps and flats: that's Lesson 2 (the five qualities). Sources: MT21C 8.1, Open Music Theory 1.1.
